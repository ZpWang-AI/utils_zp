# Utils_zp

Utilities used by zpwang.

~~~sh
# git clone repo
git clone --depth 1 https://github.com/ZpWang-AI/utils_zp.git
# or
git clone --depth 1 git@ssh.github.com:ZpWang-AI/utils_zp.git

# install
pip install -e utils_zp

# show new cmds
zp
~~~
